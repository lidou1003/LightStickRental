點擊登出系統，會跳出小視窗再三向你確認是否要登出。登出後會再回到登入系統的頁面，現在要改成登入使用者的帳號!

<br>
成功登入後就會自動跳到選擇手燈的頁面，如果在沒有選擇手燈的情況下直接點擊⌈我就要租這支手燈!!⌋，系統會提醒你需要先選擇手燈。
<img width="722" height="475" alt="image" src="https://github.com/user-attachments/assets/346c9a2a-560b-4e10-8682-5cb49358c21f" />
<br>
當你選擇了其中一隻手燈後，右側會跳出租借者須填寫的資料專區(姓名、預計租借的日期、租借天數)，填寫完畢後點擊下方⌈確認租借⌋，系統會自動幫你存檔!

<br>
現在登入的是使用者的帳號，所以點擊系統功能後不會看到⌈變更實時租金⌋，只看的到登出系統和查看歷史紀錄。
<img width="785" height="473" alt="image" src="https://github.com/user-attachments/assets/580b118f-ac0a-467d-af40-4b4fbba4a82b" />
<br>
點擊⌈查看歷史紀錄⌋後，頁面會自動跳轉到使用者租借的歷史紀錄供使用者查詢。
